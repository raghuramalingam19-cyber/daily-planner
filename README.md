# 📋 Daily Planner

A full stack web application to manage your daily tasks.

## 🌍 Live Demo
- Frontend: https://daily-planner-delta-eb.vercel.app
- Backend API: https://daily-planner-backend-tfir.onrender.com/docs

## 🛠️ Technologies Used
- **Frontend:** ReactJS
- **Backend:** Python FastAPI
- **Database:** MongoDB Atlas
- **Hosting:** Vercel (frontend) + Render (backend)

## ✨ Features
- Add tasks with due dates
- Mark tasks as complete/incomplete
- Delete tasks
- Overdue task highlighting
- Data saved permanently in cloud
